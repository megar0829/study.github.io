# 과목 소개

1. 디지털 공학

학습목표

  • 디지털 값과 아날로그 값의 변환을 공부하고 변환한 디지털 코드 값으로 논리회로를 구성할 수 있다.
  • 디지털 값에 대한 부울대수 식에 대해 공부하고 카르노 맵, 퀸-맥클러스키 공식을 이용해서 출력 부울대수 식을 최소화 하는 과정에 대해서 이해한다.

2. HTML5

학습목표

  • HTML5 문서를 작성하기 위해서 인터넷 , WWW , http 등 프로그래밍 및 웹 페이지의 대해서 이해하고 클라이언트 서버의 역할,
    웹 서버와 클라이언트 간의 데이터 교환의 과정을 이해할 수 있다.
  • html 요소와 속성을 이용해서 값을 입력하고 입력한 값이나 화면에 보여지는 요소들을 CSS 요소를 이용해서 설정할 수 있다.

3. 임베디드 시스템

학습목표

  • 아두이노를 동작시키기 위해서 C언어의 개념을 알고 아두이노 키트가 아닌 ThinkerCad 페이지를 이용해서 아두이노에 대해 이해하고 여러 종류의 센서와 연결할 수 있다.
  • 직접 아두이노 보드와 센서에 맞게 연결을 하고 코드를 작성하여 아두이노를 동작시킬 수 있다.

4. 전자회로실험I

학습목표

  • 회로를 구성하는데 필요한 소자 ( 다이오드 , 트랜지스터 , 저항 등) 에 대해서 실제 회로 구성을 어떻게 하는지 배우고 실험에 필요한 장비의 사용법에 대해서 이해하고 직접 장비를 사용할 수 있다.
  • 전자회로에서 배우는 각 소자의 회로법칙을 이해하고 실제로 적용해서 증폭기를 만들 수 있다.

5. 반도체 소자

학습목표

  • 전자와 정공이 반도체 내에서 캐리어로써 어떤 역할을 하는지 이해하고 p형 반도체, n형 반도체, 진성 반도체 에서의 캐리어에 대해서 알 수 있다.
  • 반도체에 빛을 입사 하거나 전계를 걸어 주었을 때 발생하는 확산전류, 표동 전류에 대해서 이해하고 값을 구할 수 있다.

6. 전자회로

학습목표

  • 다이오드, 트랜지스터가 회로에서 작동할 때 적용되는 전압, 전류에 대한 소자법칙을 이해하고 회로가 변함에 따라서 바뀌는 값들을 구할 수 있다.

7. 자동제어

학습목표

  • 실생활에서 존재하는 자동제어의 종류에 대해서 알고 자동제어의 동작 과정에 대해 알 수 있다.
  • 자동제어에서 활용하는 수학적 내용을 복습을 통해 이해하고 자동제어 회로에 적용시킬 수 있다.
  • 자동제어 시스템의 논리회로도를 그릴 수 있고 간단한 그림으로 나타낼 수 있다.
